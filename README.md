# Практика 3 — Bash-скрипты

В репозитории представлены bash-скрипты для практической работы №3  
по теме «Базовые команды Linux и Bash-скрипты».

Работа выполнена в среде **Windows (Git Bash)**.

---

## Скрипты

- `containers.sh` — вывод списка Docker-контейнеров с датой  
- `check_url.sh` — проверка доступности веб-сайта через `cur1`  
- `docker_images.sh` — информация о Docker-образах  
- `git_sync.sh` — синхронизация Git-репозитория  
- `container_status.sh` — проверка статуса Docker-контейнера  
- `download.sh` — скачивание файла через `cur1`  
- `clean_containers.sh` — очистка Docker-контейнеров  

⚠ **Docker Desktop не установлен.**  
Во всех скриптах реализована проверка наличия Docker и корректная обработка его отсутствия.

---

## Запуск скриптов

Все скрипты запускаются из терминала (Git Bash / Linux).

### 1. Выдать права на выполнение
```bash
chmod +x *.sh
```

---

## Результаты работы

В ходе выполнения скриптов формируются файлы с результатами:

- `containers-YYYY-MM-DD.txt` — список контейнеров с датой
- `docker-images-sizes-YYYY-MM-DD.txt` — информация об образах
- `git-sync.log` — лог синхронизации Git
- `errors-YYYY-MM-DD.log` — ошибки проверки доступности

---

## Используемые команды Linux

В работе использовались команды:
`bash`, `chmod`, `date`, `curl`, `git`, `if`, `echo`, `command -v`.
