# Практика 3 — Bash-скрипты

В репозитории представлены bash-скрипты для практической работы №3.

## Скрипты:
- containers.sh — список контейнеров (с проверкой Docker)
- check_url.sh — проверка доступности сайта
- docker_images.sh — информация об образах Docker
- git_sync.sh — синхронизация Git-репозитория
- container_status.sh — проверка контейнера
- download.sh — скачивание файла через curl
- clean_containers.sh — очистка контейнеров

⚠ Docker Desktop не установлен. В скриптах реализована обработка его отсутствия.
